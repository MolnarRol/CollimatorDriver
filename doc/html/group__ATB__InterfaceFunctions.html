Check if requested ticks (time) have passed.

Parameters
ref_ticks_U32Ticks reference for check.
checked_ticks_passed_U32Required number of ticks elapsed. To calculate this value from time you can use this macros:
  • ATB_US_TO_TICKS_dM_U32()
  • ATB_MS_TO_TICKS_dM_U32()
  • ATM_S_TO_TICKS_dM_U32()
Returns
Time passed status.
Return values
1number of ticks have passed
0number of ticks have not passed

time1_U32 = ATB_GetTicks_U32(); // Write new reference time
}
}
